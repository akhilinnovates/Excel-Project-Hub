# Bike Buyers Sales Analysis Dashboard

## Project Overview

This project analyzes customer purchasing behavior in the bike market using Microsoft Excel. The dataset was cleaned and transformed to ensure data accuracy before performing exploratory data analysis. Interactive dashboards were developed to provide meaningful insights into customer demographics and purchasing trends.

## Key Features

* Data Cleaning and Preprocessing
* Pivot Tables for Data Analysis
* Interactive Dashboard Creation
* Dynamic Slicers for Filtering Data
* Customer Demographic Analysis
* Visual Insights through Charts and Graphs

## Analysis Performed

* Relationship between age groups and bike purchases
* Regional distribution of bike buyers
* Impact of education level on purchasing decisions
* Analysis based on marital status
* Customer segmentation and trend identification

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Dashboard Design
